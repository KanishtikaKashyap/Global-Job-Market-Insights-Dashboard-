An interactive Power BI dashboard built using a cleaned global job market dataset. The project focuses on Exploratory Data Analysis (EDA), data cleaning, and visualization to uncover hiring trends, top companies, job types, experience requirements, skills, and locations.

---

## Project Overview

This project analyzes global job postings to answer key business questions such as:

- Which companies hire the most?
- Which skills are most demanded?
- Which locations have the highest number of jobs?
- What experience levels are most required?
- Which job types are most common?

The dashboard enables users to interactively explore hiring trends using filters and visualizations.

## Data Cleaning

The dataset was cleaned using Python and Power Query.

Cleaning steps included:

- Removed duplicate records
- Removed unnecessary columns
- Removed extra spaces
- Renamed column names
- Standardized text values
- Handled missing values
- Corrected data types
- Created clean dataset for visualization

## Key Insights

- 465 total job postings analyzed.
- 382 unique job titles.
- 35 companies included.
- 46 hiring locations identified.
- Experienced professionals account for the largest share of job postings.
- Sales and R&D are among the most frequently requested skills.
- Full-time jobs dominate the dataset.
